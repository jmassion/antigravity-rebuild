HOW TO DEPLOY:

  This folder is structured for direct deployment —
  index.html is at the root for proper domain resolution.

  Option A: Deploy via Vercel (garfield.holodeckos.com)
  Option B: Drag the root folder onto netlify.com
  Option C: Upload root contents via FTP to any host
  Option D: Open index.html locally in a browser

  No build step. No npm install. No configuration needed.
