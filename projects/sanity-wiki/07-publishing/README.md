# 📡 Publishing & Distribution
Publishing channels, workflows, and distribution strategy.

## Channels
| Channel | Purpose | Status |
|---------|---------|--------|
| Vercel (Web) | Primary web hosting | ✅ Active |
| GitHub | Source code & collaboration | ✅ Active |
| SiteGround | DNS & domain management | ✅ Active |

## Publishing Workflow
1. Make changes locally
2. Commit with descriptive message
3. Push to `main` branch
4. Vercel auto-deploys (~30s)
5. Verify at live URL
