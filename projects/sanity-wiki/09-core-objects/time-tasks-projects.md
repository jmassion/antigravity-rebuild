# ⏰ Time, Tasks & Projects
Work management objects.

## Time
- Schedules, deadlines, milestones
- Time tracking per task/project
- Calendar integration (future)

## Tasks
| Field | Description |
|-------|-------------|
| Title | Task name |
| Status | Todo, In Progress, Done |
| Assignee | Character or person |
| Project | Parent project |
| Priority | High, Medium, Low |
| Due Date | Deadline |

## Projects
Collections of tasks with overarching goals.

## Processes
Repeatable workflows that can be templated and reused.
