# 🧩 Core Objects
The "playset" — all object types that model the real and virtual world.

## Overview
Think of this as organizing a playset that simplifies how we model everything. Each object type maps to a Sanity Studio schema (future) and a 3D visualization (via NanoBanana/Meshy).
