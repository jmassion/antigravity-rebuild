## Agent Template

When adding a new agent, document it with:

```markdown
### Agent Name

- **Role**: Builder / Reviewer / Coordinator / Specialist
- **Model**: Claude Opus 4.6 / GPT-4o / etc.
- **Skills**: List of skills from [[03-skills]]
- **Rules**: Link to agent-specific rules in [[05-rules]]
- **Workstation**: Virtual workspace description
- **Status**: Active / Paused / Retired
```
