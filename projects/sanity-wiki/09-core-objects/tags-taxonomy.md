# 🏷️ Tags & Taxonomy
How everything is categorized and found.

## Tag System
Tags are lightweight labels applied to any object type.

## Categories
| Category | Tags |
|----------|------|
| **Type** | agent, skill, rule, brand, project, task |
| **Status** | active, draft, archived, planned |
| **Priority** | high, medium, low |
| **Domain** | design, code, content, infrastructure |

## Taxonomy Hierarchy
Tags can be nested: `design/3d`, `code/frontend`, `content/wiki`.
