# 💬 Chat Logs

Saved conversations from AI agent sessions.

## How Chat Logs Work

Agent conversations are saved as markdown files in this folder, organized by date and topic.

## Naming Convention

```
YYYY-MM-DD_topic-slug.md
```

Example: `2026-04-10_wiki-setup.md`

## Template

```markdown
---
date: 2026-04-10
agent: antigravity-main
topic: Wiki Setup
tags: [setup, wiki, infrastructure]
---

# Chat: Wiki Setup

## Summary
Brief summary of what was discussed/accomplished.

## Key Decisions
- Decision 1
- Decision 2

## Transcript
Full or summarized conversation...
```
