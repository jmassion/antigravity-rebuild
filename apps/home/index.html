      About AntiGravity

      AntiGravity is a running collection of prototypes — platforms, 3D experiments, wikis, data browsers, and tools — built rapidly with AI agents and rebuilt here as one navigable, permanently hosted portfolio. Every project is a path on a single origin: no subdomains, no servers, no tokens.

      How it works

      A single catalog.json drives everything — the gallery, the 3D views, screenshots, and deploys. Static copies and compiled builds live under /p/<slug>/; GitHub Actions rebuilds and ships on every push.
