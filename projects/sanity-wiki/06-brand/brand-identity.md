# 💎 Brand Identity

Core brand identity guidelines for AntiGravity and its sub-brands.

## Brand Architecture

```
AntiGravity (Parent)
├── HolodeckOS — Platform umbrella & personal OS
├── Sanity Wiki — Project knowledge base
├── FranchiseOS — Franchise management system
├── AlphaUnicorn — Business consultancy & education
└── Paperclip — AI agent orchestration platform
```

| Element | Value |

| **Name** | AntiGravity |

## Design Language

### Liquid Glass Aesthetic
The signature visual style — see [[04-styles/liquid-glass]] for full specification.

- Frosted glass panels with subtle blur
- Depth layers and ambient glow
- Dark mode as default, light mode available
- Gradient accents (purple → teal)

### Color Palette
See [[04-styles/color-palette]] for complete token system.

| Role | Dark Mode | Light Mode |
|------|-----------|------------|
| Primary Accent | `#7C6AFF` | `#6C5CE7` |
| Secondary Accent | `#00D4AA` | `#00B894` |
| Tertiary Accent | `#FF6B9D` | `#E84393` |

### Typography
See [[04-styles/typography]] for complete font system.

| Usage | Font | Weight |
|-------|------|--------|
| Headings | Outfit | 700–800 |
| Body | Inter | 400–500 |
| Code | JetBrains Mono | 400–500 |

## Voice & Tone

- **Clear over clever** — explain simply, avoid jargon
- **Confident, not arrogant** — state facts, acknowledge unknowns
- **Forward-looking** — emphasize possibilities and vision
- **Inclusive** — welcome all skill levels

## Logo Usage

Logo assets are stored in `06-brand/assets/`. Guidelines:
- Minimum clear space: 1x logo height on all sides
- Never stretch, skew, or recolor without approval
- Use on dark backgrounds when possible for best impact
