# 🎨 Styles

Design system, visual guidelines, and 3D visualization pipeline.

## Overview

The AntiGravity visual language is built on the **Liquid Glass** aesthetic — frosted glass panels with depth, subtle blur, and premium cinematic quality. Think Unreal Engine 5 AAA game UI rendered with RTX 5090 quality.

## Design Principles

1. **Cinematic Quality** — Every screen should feel like a AAA game interface
2. **Clean White Rooms** — Empty white backgrounds with hyperrealistic content
3. **Glassmorphism** — Frosted glass panels with depth and layers
4. **Characters Match Roles** — Virtual team members styled for their function
5. **Visuals Over Text** — Hero images and 3D scenes communicate before words
6. **Micro-Animations** — Subtle hover effects and transitions everywhere
7. **Premium Feel** — Nothing should look like a minimum viable product
