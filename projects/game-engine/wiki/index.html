2 World — Where Everything Lives

The space itself. What's in it. How it looks. Everything is a Thing.

Thing {
  name: "My Chair"
  kind: "furniture"
  placement: Placement       // Where it is
  shape: Shape               // Geometry
  surface: Surface           // Visual appearance
  children: [Thing, ...]     // Nested Things
  traits: [Trait, ...]       // Behaviours
  data: {}                   // Data binding
}

3 Senses — Listening to Every Device

The universal input layer. Every device speaks differently, but the engine hears one language.

The 3D Pointer

A pointer that exists in 3D space, not just on a 2D screen. Modes: free, snap, surface, gravity.

Device How the Pointer Moves
Mouse Raycast into 3D. Scroll = depth.
Touch One finger = pointer. Pinch = zoom. Long press = grab.
Voice "Move it left." "Put it on the table."
Gamepad Left stick = XY. Right stick = camera. Triggers = depth/grab.
Gyroscope Tilt to look. Roll to rotate pointer.
Stream Deck Buttons trigger macro actions.

4 Life — Physics, Gravity, and Behaviour

Instead of complex component hierarchies, Things have Traits — simple, stackable behaviour modules.

Floats Spins Follows Breathes Glows Reacts Speaks Thinks

let lamp = new Thing("Magic Lamp")
lamp.add(new Glows({ color: "warm-gold" }))
lamp.add(new Breathes({ speed: "slow" }))
lamp.add(new Reacts({ on: "pointer-near", do: "glow-brighter" }))

SpaceML — 3D Markup Language

Human-readable markup for describing 3D scenes. Like HTML for holographic spaces.

space "My Living Room"
  sky: warm-sunset
  gravity: earth

  thing "Coffee Table"
    shape: box
    size: 1.2m x 0.5m x 0.6m
    surface: oak-wood

    thing "Laptop"
      shape: import "macbook.gltf"
      position: on-top, center
      traits:
        - glows: screen-blue, soft

  portal "To Garden"
    leads-to: "Garden Space"
    preview: on
