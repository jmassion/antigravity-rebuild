✏️The Editor

A VS Code-based environment with rich AI enhancements — Tab autocomplete, inline Commands, and the Agent side panel.

Your Crystalline Workspace

The AntiGravity editor is built on VS Code, enhanced with AI superpowers that transform how you write code. Think of it as Lumi's crystalline desk — familiar tools made magical.

AI-Enhanced Features

⚡ Tab — Advanced Autocomplete

Like having Ribbit whisper the next line of code. Tab provides intelligent, context-aware completions that understand your entire codebase.

💬 Command — Inline Instructions

Press Cmd + K to open the inline command bar. Describe what you want in natural language, and the editor transforms your words into code.

🐸 Agent Panel

The side panel where your full AI agent lives. For complex, multi-step tasks that go beyond simple edits — research, refactoring, debugging, and more.

💡 Pro Tip

Use Tab for quick completions, Command for single-file edits, and the Agent Panel for complex multi-file operations. Each tool has its sweet spot!
