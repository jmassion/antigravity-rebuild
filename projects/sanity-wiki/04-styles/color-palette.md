# 🎨 Color Palette

Color tokens for the AntiGravity design system.

## Primary Palette

| Token | Hex | Usage |
|-------|-----|-------|
| `--accent-primary` | `#6C63FF` | Primary actions, active states |
| `--accent-secondary` | `#00D4AA` | Success, positive states |
| `--accent-tertiary` | `#FF6B9D` | Alerts, featured items |
| `--accent-warning` | `#FFB84D` | Warnings, pending states |
| `--accent-error` | `#FF4757` | Errors, destructive actions |

## Neutral Palette

| Token | Hex | Usage |
|-------|-----|-------|
| `--bg-deep` | `#0A0A0F` | Page background |
| `--bg-surface` | `#12121A` | Card surfaces |
| `--bg-elevated` | `#1A1A25` | Elevated elements |
| `--border-subtle` | `rgba(255,255,255,0.08)` | Subtle borders |
| `--border-default` | `rgba(255,255,255,0.12)` | Default borders |

## Gradients

```css
--gradient-primary: linear-gradient(135deg, #6C63FF, #00D4AA);
--gradient-warm: linear-gradient(135deg, #FF6B9D, #FFB84D);
--gradient-cool: linear-gradient(135deg, #00D4AA, #6C63FF);
--gradient-glass: linear-gradient(135deg, rgba(255,255,255,0.1), rgba(255,255,255,0.02));
```
