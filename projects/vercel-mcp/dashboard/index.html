📋 SiteGround DNS Quick Guide

How to Add an A Record in SiteGround

  1. Log in to SiteGround Client Area → click Websites
  2. Find holodeckos.com → click Site Tools
  3. In the sidebar, go to Domain → DNS Zone Editor
  4. Select A from the record type dropdown
  5. Enter the subdomain prefix in Name (e.g., noahos)
  6. In Address, paste: 76.76.21.21
  7. Leave TTL at default or set to 3600
  8. Click Create — wait 5-30 min for propagation
  9. Verify by visiting subdomain.holodeckos.com in your browser
