# 🔤 Typography

Font system and typographic hierarchy.

## Fonts

| Font | Weight | Usage |
|------|--------|-------|
| **Outfit** | 600, 800 | Headlines, section titles |
| **Inter** | 300, 400, 500, 600 | Body text, UI elements |
| **JetBrains Mono** | 400 | Code blocks, technical content |

## Load via Google Fonts

```html
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@300;400;500;600&family=Outfit:wght@600;800&family=JetBrains+Mono:wght@400&display=swap" rel="stylesheet">
```

## Type Scale

| Level | Font | Size | Weight | Usage |
|-------|------|------|--------|-------|
| H1 | Outfit | 2.5rem | 800 | Page titles |
| H2 | Outfit | 1.75rem | 600 | Section headers |
| H3 | Inter | 1.25rem | 600 | Subsection headers |
| Body | Inter | 1rem | 400 | Paragraphs |
| Small | Inter | 0.875rem | 400 | Captions, metadata |
| Code | JetBrains Mono | 0.875rem | 400 | Code snippets |
