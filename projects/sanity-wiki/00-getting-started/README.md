## Key Principles

- **Everything is a file** — Markdown-first, git-tracked, agent-readable
- **Backlinks matter** — Every `[[wiki-link]]` is tracked bidirectionally
- **Agents coordinate** — The [[02-agents/orchestration]] system prevents conflicts
- **Visuals communicate** — Use hero images, diagrams, and 3D scenes alongside text
- **Future-ready** — Structure is designed to migrate into [[11-planning/sanity-studio]]
