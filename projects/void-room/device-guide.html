🚀 Step Zero: Host Void Room Online

Before ANY device can access Void Room, you need it hosted at a URL. It's a single HTML file — here are the easiest ways to get it online for free:

⚡ Netlify Drop

Drag and drop the HTML file. Live in seconds. Free. No account needed for a quick test.

1. Go to app.netlify.com/drop 2. Drag void-room.html onto it 3. Get your URL instantly!

🔺 Vercel

Push to GitHub, auto-deploys. Free tier is generous. Great for permanent hosting.

npm i -g vercel vercel deploy --prod

📄 GitHub Pages

Free hosting straight from a GitHub repo. Push the file, enable Pages in settings.

1. Create repo, push HTML file 2. Settings → Pages → main branch 3. Live at username.github.io/repo

🌊 DigitalOcean

Your existing droplet! Just serve the file with nginx, caddy, or even Python.

cd /var/www/html # copy void-room.html here python3 -m http.server 8080

☁️ Cloudflare Pages

Blazing fast global CDN. Free tier. Connect to Git or direct upload.

1. dash.cloudflare.com → Pages 2. Upload void-room.html 3. Instant global deployment

🏠 Local Network

For home-only use: run a simple server on any computer. All local devices can connect.

# From the folder with the file: python3 -m http.server 8080 # Open http://YOUR-IP:8080
